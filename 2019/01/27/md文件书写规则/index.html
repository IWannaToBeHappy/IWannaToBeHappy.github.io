md文件书写规则

发表于 2019-01-27 | |
字数统计: 810

前言

Hexo、CSDN以及简书中的文章都是使用markdown文章编写的，也就是，编写在md文件中，md文件所遵循的书写规则如下:
原文链接：https://www.jianshu.com/p/de9c98bba332

markdown语法

  • 标题

    • 一级标题

      # 一级标题

    • 二级标题

      ## 二级标题

    • 三级标题

      ### 三级标题

    • 四级标题

      #### 四级标题
    • 五级标题
      ##### 五级标题
    • 六级标题
      ###### 六级标题

  • 引用

    > 引用内容

    可以重复引用

  • 段落

    回车触发段落划分

  • 分隔符

    连续三个及以上的短横杠 ---

  • 列表

  1. 无序列表

    使用 * - + 中的任意一个符号加空格创建无序列表，其中 + - 组合起来可以创造具有层次关系的有序列表，直接使用 * 也可以

  1. 有序列表
    使用1. 内容创建有序列表

  • 链接

    可以用文字附加给文字超链接

    也可以直接用尖括号构造链接 https://iwannatobethegay.github.io

  • 图片

    与链接相似，使用的是 ![名称](地址) 格式

  • 代码段

    • 使用 单个重音号（位于！左边）包围代码，形成代码块
    • 使用 三个重音号（位于！左边）包围代码，形成代码段
    • 代码段的前三个重音号后面可以写入不包含空格的注释，空格后可以写入代码框标题

  • 字体样式

    • 倾斜 *倾斜*或者 _倾斜_
    • 加粗 **加粗**
    • 倾斜并加粗 ***倾斜并加粗***
    • 删除 ~~删除~~
  • 表格
    • 使用 - | 符号将内容分割
    • 使用 : 符号标识对齐
    • 暂时不知道如果想在表格中表示|符号，要怎么办
    • 使用
      实现表格内换行
      表头1 表头2 表头3 表头4
      默认左对齐 左对齐 居中对其 右对齐
      默认左对齐 左对齐 居中对其 右对齐
      默认左对齐 左对齐 居中对其 右对齐

矩阵输入

  • 简单矩阵输入

$$
\begin{matrix}
1&2 \ 3&4 \ 5&6
\end{matrix}
$$

  • 带大括号矩阵

$$
\left{
\begin{matrix}
1&2 \ 3&4 \ 5&6
\end{matrix}
\right}
$$

  • 带中括号矩阵

$$
\left[
\begin{matrix}
1&2 \ 3&4 \ 5&6
\end{matrix}
\right]
$$
